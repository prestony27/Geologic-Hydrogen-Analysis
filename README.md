# Geologic Hydrogen Analysis

Analysis and visualization of USGS geologic hydrogen prospectivity data with congressional district overlays.

## Data Sources

- **USGS Hydrogen Prospectivity**: [FeatureServer](https://services.arcgis.com/v01gqwM5QqNysAAi/arcgis/rest/services/Naturally_Occurring_Geologic_Hydrogen_Prospectivity_Maps_10_22_2024_WFL1/FeatureServer)
- **ScienceBase Data Release**: [Download](https://www.sciencebase.gov/catalog/item/671be1c2d34efed56210dded)
- **Congressional Districts**: [Census TIGERweb](https://tigerweb.geo.census.gov/arcgis/rest/services/TIGERweb/Legislative/MapServer)

## Layers Available

- Total Prospectivity
- Source: Serpentinization, Radiolysis, Deep/Mantle
- Reservoir
- Seal
- Known Hydrogen Occurrences
